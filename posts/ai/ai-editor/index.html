Gemini cli#

安装nodejs。

1

brew install node

安装 Gemini CLI 

1

npm install -g @google/gemini-cli
  • 权限：默认可访问 Gemini 2.5 Pro 模型，支持最大 1M token 的上下文窗口。
  • 计费：免费额度为每日 1,000 次请求。
  • 说明：推荐登录时使用 Google 账户，无需管理 API 密钥，适合个人开发和试用。
  • 重要：最主要这不会占用你的免费 API 额度！

启动：

1

gemini

或者指定模型：

1

gemini -m "gemini-2.5-pro"

如果要写项目代码，一定要进入项目目录。
