npm install 报错#

不要看前边关于python的报错，不需要安装python。

直接拉到最后看报错：

1

Failed at the node-sass@6.0.1 postinstall script.

再结合网上的说法，推测是node-sass的问题。执行如下：

1

npm i node-sass --sass_binary_site=https://npm.taobao.org/mirrors/node-sass/

再次执行npm install，无报错，执行npm run dev，项目启动成功。
