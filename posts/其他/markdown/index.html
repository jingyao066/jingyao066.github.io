文本样式#

  • 加粗
    字体两边各加两个星号 ** 快捷键ctrl+b

  • 斜体
    字体两边各加 __ 快捷键 ctrl+i

  • 删除线
    各加两个波浪线。 快捷键：ctrl+u ,和uc浏览器有热键冲突

  • 分段
    分段：段落之间空一行即可。

换行#

1. 普通换行（软换行）
在行尾添加 两个空格，然后按回车键。（注意：空格在渲染后不可见，但会触发换行）

1
2

这是第一行（末尾两个空格）  
这是第二行

效果：
这是第一行
这是第二行

2. 段落换行（硬换行）
直接空一行（按两次回车），会创建一个新段落（段落间有默认间距）。

1
2
3

这是第一段

这是第二段

效果：
这是第一段

这是第二段

3. HTML 换行（兼容性更强）
使用 HTML 的 <br> 标签（某些严格解析的 Markdown 编辑器可能需要此方式）：

1
2

这是第一行<br>
这是第二行

效果：
这是第一行

这是第二行

列表#

  • 无序列表
    使用星号(*)、加号(+)或是减号(-)作为列表标记，注意符号后边一定要跟一个空格。
    在星号前加两个空格键或者一个tab键就可以产生一个子列表。

  • 有序列表
    在文本前加字母或数字即可生成一个有序列表。
    注意，当你第一个序号使用什么作为标记的，那么同级别的列表就会自动使用其作为标记。

锚点#

在需要跳转的位置添加锚点，语法如下：
<span id="jump">跳转到的地方</span>

在需要点击跳转的位置，使用上面的id，格式类似超链接的形式：
[点击跳转](#jump)
